# Content Strategy & Marketing Playbook

This document defines the core content loops and marketing funnels used by Antharmaya Labs to build audience, authority, and lead flow.

## 1. The Core Narrative: "Software Engineering in 2026"
The primary top-of-funnel narrative revolves around providing extreme value to developers trying to break into or survive in big tech/startups.
*   **The Hook:** "I broke into big tech by doing everything in this roadmap and more for 223 days. Since then I got the return offer at Amazon, then turned it down to build my own VC backed startup. So safe to say it works. Here’s everything I would do if I had to restart so you can copy me."
*   **The Content Pillars:**
    *   Learn Python with Harvard CS50.
    *   Build real end-to-end projects, one per week.
    *   Learn AI/ML concepts (the future).
    *   Master system design and version control.
    *   Prep for SWE interviews with DSA (Neetcode).
*   **The Call to Action (CTA):** The resume is the bottleneck. It must be ATS-friendly. Point them to a productized service or tool (e.g., "ResuMax" concept).

## 2. Weekly Publishing Rhythm (Antharmaya Labs)
Consistency is the only metric that matters at the start.
*   **Cadence:** Minimum 1 post per week across core channels.
*   **Channels:**
    *   **X (Twitter):** Short-form insights, build-in-public snippets, technical wins.
    *   **LinkedIn:** Professional development, the SWE 2026 narrative, long-form roadmaps.
    *   **Reddit (r/antharmayalabs + niches):** Deep-dive technical explanations, helping others debug.
    *   **Blog / Newsletter:** Consolidating weekly learnings into a permanent archive.

## 3. The "Build in Public" Ethos
*   Share the failures, the architectural pivots, and the revenue milestones (once hit).
*   Documenting the build process of **PhotoSelect** and the Horizon OS upgrades creates a moat of transparency and trust.

## 4. The "Second Brain" Marketing Engine (Obsidian + GHL)
The content strategy is powered by a dual-platform system connecting knowledge to execution.
*   **Obsidian (The Idea Engine):** Where P.A.R.A. lives. This is where roadmaps, technical deep-dives, and content drafts are born.
*   **GoHighLevel (The Execution Arm):** The CRM that catches the traffic generated by the content. Funnels, email sequences, and lead tracking happen here.
*   **The Bridge:** When content (from Obsidian) drives a sign-up, that lead enters GHL. Zapier/Make scripts (or Claude Code automation) ensure that lead data and campaign metrics flow back into Obsidian for strategic review in Horizon OS.
