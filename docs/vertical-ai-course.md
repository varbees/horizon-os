# Vertical AI Course Map

## Chosen Wedge

Stay in one lane: vertical AI for Indian SMBs, but do not activate all verticals at once.

The best course is:

1. PhotoSelect Revenue Mode
2. Wedding Studio AI Ops Agent
3. Antharmaya Agents as internal infrastructure
4. Coaching centre AI ops after the studio wedge repeats
5. Clinic/diagnostic journey agent only after paid discovery
6. AI ops observability after real agent workloads exist

## Why

PhotoSelect already gives a real buyer surface: Indian wedding and event studios. The first move is monetization and implementation, not more code. The studio AI ops agent is the closest adjacent upsell because it uses the same customer trust and solves lead, follow-up, payment, and delivery friction.

## Active Now

### PhotoSelect Revenue Mode

- Buyer: mid-size Indian wedding and event studios doing 15-30 events per month.
- Offer: paid implementation package plus ongoing subscription.
- Outcome: raw card to client-approved selects in 48 hours.
- Calendar: protected in Deep Work 2 every weekday.

### Wedding Studio AI Ops Agent

- Buyer: existing or warm PhotoSelect studios.
- Offer: lead capture, follow-up, quote breakdowns, payment nudges, and selection reminders.
- Outcome: fewer missed leads, faster payments, cleaner handoffs.
- Calendar: only inside Income Engine or PhotoSelect pilot blocks until sold.

## Internal Platform

Antharmaya Agents is not the first thing sold. It is built as delivery infrastructure after two or three paid implementations repeat the same workflow.

Shared modules:

- Lead database
- Conversation history
- Reminder engine
- WhatsApp/email adapters
- Mini CRM
- Analytics and failure logs

## Parked But Valid

- Coaching Centre AI Ops: strong later vertical because inquiries, batches, fee reminders, and parent communication are repeatable.
- Clinic / Diagnostic Journey Agent: valuable but higher trust and privacy burden.
- AI Ops Observability: useful after Antharmaya runs enough real agent workloads.

This gives a shorter public scoreboard without distracting from service revenue.
