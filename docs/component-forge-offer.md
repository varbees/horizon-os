# Interactive Component Forge Offer

## Positioning

I ship high-polish interactive product surfaces for B2B SaaS teams that need AI features, dashboards, automation views, or complex workflows to feel real quickly.

## Core Offer

One week design-engineering sprint:

- Audit one workflow
- Build one interactive React prototype
- Add motion and responsive behavior
- Package the component as reusable code
- Deliver a short implementation memo

## First Outreach Asset

Send a specific prospect a tiny visual improvement or component concept based on their product. The goal is not a generic portfolio link. The goal is to make the buyer feel that you already understand their interface.

## Starter Components

- SaaS metric command deck
- AI workflow inspector
- Sales automation canvas
- Media approval timeline
- Pricing capacity calculator
