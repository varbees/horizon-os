# Journey Log System

The logbook is the Obsidian-grade memory layer for the next two years. It records evidence, places, decisions, body progress, project proof, money movement, and mental clarity without turning the app into a diary.

## Entry Types

- Field Scout: place, GPS, altitude, access, photos, terrain, difficulty, next visit.
- Body Proof: movement session, hill/trek capacity, recovery, pain, endurance marker.
- Build Proof: shipped screen, commit, deployment, user conversation, demo link.
- Capital Proof: income, expense, target progress, runway change.
- Spec Proof: land, water, power, shelter, food, legal, internet, budget item.
- Reflection: short note from the daily or weekly review.

## Required Fields

- Date and timezone.
- Anchor: Body, Attention, Capital, Spec, Product, Service, or Public Proof.
- Evidence: link, path, coordinates, screenshot, command output, or commit.
- Lesson: one sentence.
- Next action: one physical action.
