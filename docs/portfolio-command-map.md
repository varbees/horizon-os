## Resurrection Gate

A parked or archived project can become active only if all five answers are concrete:

1. Who pays?
2. What painful job do they already spend money or time on?
3. Can the MVP ship in two weeks?
4. Can you reach 25 buyers this week?
5. Does it strengthen PhotoSelect, Antharmaya, Component Forge, Agent Linux Control, or RateGuard?

If any answer is vague, the project stays parked.
